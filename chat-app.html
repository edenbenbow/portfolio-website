Chat App

Github | Kanban board

Objective

To build a React Native mobile chat application, using Agile project-management techniques including: user stories, story points, and a Kanban board.

Description

Chat App is a React Native chat application that lets the user share their location, add and share photos, compose messages offline, and use screenreaders in addition to other features.

Technologies used

  • React Native
  • Expo
  • Google Firebase
  • Android Emulator
  • Gifted Chat
  • Real-time data
  • React Native asyncStorage API
