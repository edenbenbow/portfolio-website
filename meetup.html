Meetup

Demo the app | Github

Objective

To build a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique.

Description

Meetup is a serverless, progressive web application whose features include geolocation, city search, visualizations, and the ability to work offline.

Technologies used

  • React
  • Recharts
  • Puppeteer
  • Lighthouse
  • Meetup API
